# php-websocketd
Turn any program that uses STDIN/STDOUT into a WebSocket server and display in the browser.

# Start and stop
**start**  
php start.php start -d

**stop**  
php start.php stop

# Change command
Open ```Applications/php-websocketd/start.php ``` and change CMD constant.
